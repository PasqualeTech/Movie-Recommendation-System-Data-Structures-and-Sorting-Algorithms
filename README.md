# Movie Recommendation System: Data Structures & Algorithms

[Italiano](#italiano) | [English](#english)

---

## <a name="italiano"></a> 🇮🇹 Versione Italiana
Un sistema di suggerimento film che utilizza le mappe (`HashMap`) per organizzare i contenuti e algoritmi di selezione casuale per fornire raccomandazioni personalizzate all'utente.

### 🛠️ Competenze Tecniche
* **Advanced Data Structures:** Implementazione di `HashMap` per l'indicizzazione dei dati.
* **Search Algorithms:** Logica di filtraggio basata su input utente.
* **Randomization:** Uso della classe `Random` per simulare raccomandazioni dinamiche.

---

## <a name="english"></a> 🇬🇧 English Version
A movie recommendation system using `HashMap` to organize content and random selection algorithms to provide personalized user suggestions.

### 🛠️ Technical Skills
* **Advanced Data Structures:** Implementation of `HashMap` for data indexing.
* **Search Algorithms:** Filtering logic based on user preference.
* **Randomization:** Using the `Random` class to simulate dynamic recommendations.
